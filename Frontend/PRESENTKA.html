Концепция проекта

Проблема

Существующие платформы слишком формальны, не учитывают специфику поиска первой работы и стажировки. Студенты теряются в вакансиях, работодатели — в откликах.

Решение

«Трамплин» — экосистема, где студенты не просто ищут работу, а строят карьеру с нуля: находят менторов, участвуют в мероприятиях и получают предложения о стажировках на основе навыков и активности.

Целевая аудитория
  • Студенты и выпускники IT-специальностей
  • IT-компании (от стартапов до корпораций)
  • Карьерные центры вузов
Социальная значимость
  • Снижение барьера входа в IT
  • Нетворкинг и менторство
  • Безопасная модерация контента

Архитектура и структурные формы

FSD (Feature-Sliced Design)

Модульная архитектура фронтенда: разделение на слои (app, pages, features, widgets, entities, shared) обеспечивает масштабируемость и переиспользуемость кода.

Backend на Go

Высокопроизводительный REST API с JWT-аутентификацией, кэшированием в Redis и хранением в PostgreSQL. Микросервисная структура для гибкости.

Аргументация выбора

  • React + Vite — быстрая разработка, компонентный подход, отличная производительность сборки.
  • shadcn/ui — кастомизируемые компоненты, идеально подходят для тёмной темы и современного UI.
  • Go — высокая производительность под нагрузкой, отличная поддержка конкурентности, простота деплоя.
  • PostgreSQL — надёжность, поддержка JSONB для гибких схем, полнотекстовый поиск.
  • Redis — мгновенное кэширование, хранение сессий, rate limiting.
  • Yandex Maps API — качественная картографическая база по России, удобная интеграция с React.

// useFilters — кастомный хук для фильтрации
export const useFilters = (opportunities) => {
  const [selectedSkills, setSelectedSkills] = useState([]);
  const [selectedLevels, setSelectedLevels] = useState([]);

  const filteredOpportunities = useMemo(() => {
    return opportunities.filter(opp => {
      if (selectedSkills.length) {
        const oppSkills = opp.Tags?.map(t => t.name) || [];
        if (!selectedSkills.some(s => oppSkills.includes(s))) 
          return false;
      }
      if (selectedLevels.length) {
        const levelName = getExperienceLevelName(opp.ExperienceLevel);
        if (!selectedLevels.includes(levelName)) return false;
      }
      return true;
    });
  }, [opportunities, selectedSkills, selectedLevels]);

  return { filteredOpportunities, toggleSkill, resetFilters };
};

export const useNotifications = () => {
  const [notifications, setNotifications] = useState([]);
  
  const markAsRead = useCallback((id) => {
    setNotifications(prev =>
      prev.map(n => n.id === id ? { ...n, isRead: true } : n)
    );
  }, []);

  const acceptConnection = useCallback((id) => {
    setNotifications(prev => prev.filter(n => n.id !== id));
    // Отправка запроса на добавление в контакты
  }, []);

  return { notifications, unreadCount, markAsRead, acceptConnection };
};

func VerifyCompany(c *fiber.Ctx) error {
    var req VerifyRequest
    if err := c.BodyParser(&req); err != nil {
        return c.Status(400).JSON(fiber.Map{"error": "Invalid request"})
    }
    // Проверка ИНН через внешнее API + корпоративная почта
    if isValidINN(req.INN) && isValidCorporateEmail(req.Email) {
        updateCompanyStatus(req.CompanyID, "verified")
